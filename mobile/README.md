## :bookmark_tabs: Pré-requisitos

* yarn
* expo

## :wrench: Configurando

Configurar a **baseURL** do axios em src/services/api.json

```js
const api = axios.create({
  baseURL: 'baseURL',
});
```

## :inbox_tray:  Instalando os módulos

```sh
yarn install
```

## :rocket: Iniciando o projeto

```sh
yarn start
```
